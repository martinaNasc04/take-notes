# Projeto Take Notes
Projeto Take Notes é uma aplicação web desenvolvida com Next.js e TypeScript, projetada para a criação e gerenciamento de notas de forma eficiente e intuitiva.

-----
## Funcionalidades
 Criação, edição e exclusão de notas.

----
## Tecnologias Utilizadas
- Next.js
- TypeScript
- Tailwind CSS
- Lucide Icons
- Clerk
- Prisma
